## xcover wait

wait

### Synopsis

wait waits for the xcover server to be ready

```
xcover wait [flags]
```

### Options

```
  -h, --help                 help for wait
  -s, --socket-path string   Path to the xcover socket file (default "/tmp/xcover.sock")
      --timeout duration     Timeout (default 2m0s)
```
